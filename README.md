# Internship & Placement Tracking Portal

## Overview
A Java-based console application for managing students, jobs, and internship applications.

## Features
- Student Management
- Job Management
- Application Tracking
- Search Functionality
- OOP Concepts Implementation

## Technologies

- ArrayList
- OOP
- Git & GitHub

- Method Overloading
- Static Members
